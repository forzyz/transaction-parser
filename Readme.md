    Purpose:
        A simple PHP-based solution for displaying and formatting transaction data.
        Automatically computes and displays total income, expenses, and net balance.

    Key Features:
        Dynamically generated table headers based on data structure.
        Color-coded formatting for positive and negative amounts.
        Displays categorized totals in a visually clear layout.

    How to Use:
        Upload a CSV file or process a dataset through $parsedData.
        Customize formatting functions (formatDate, formatDollarAmount) as needed.

    Technologies:
        PHP for backend logic.
        HTML/CSS for frontend styling.
